# F4llTokens

F4llTokens is a simple token generator tool built in Python.

## Features

- **Token Generation:** Generate random Discord tokens.
- **Token Validation:** Check the validity of generated tokens.
- **Webhook Integration:** Optionally send valid tokens to a Discord webhook.

## Getting Started

Follow these steps to set up and run F4llTokens on your machine.

### Prerequisites

- Python 3.x

### Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/F4llDev/F4llTokens.git
   ```
2. Install Dependencies
      ```bash
      pip install -r requirements.txt
      ```

3. Run the tool either doing
   ```bash
   python main.pyw
   ```
   Or double clicking on the file main.pyw
   
### Configuration
Webhook Integration: Set up a Discord webhook to receive notifications for valid tokens. Specify the webhook URL during the tool's runtime.
